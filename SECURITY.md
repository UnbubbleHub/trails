# Security Policy

If you find a vulnerability in Unbubble Trails, please **do not** open a public
issue. Instead:

- **Preferred:** [open a private security advisory](https://github.com/UnbubbleHub/trails/security/advisories/new)
  on this repository, or
- contact the maintainers via the [UnbubbleHub organization](https://github.com/UnbubbleHub).

We aim to acknowledge reports within a few working days. Please include enough
detail to reproduce — affected version/commit, steps, and impact.

## Scope

In scope: code in this repository (route handlers, agent, repo layer, scripts).

Out of scope: vulnerabilities in third-party services we depend on (Telegram,
OpenAI, Exa, Firebase) — report those directly to the upstream vendor.
